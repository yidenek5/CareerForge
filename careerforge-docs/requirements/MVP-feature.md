# CareerForge MVP Features

## 1. Overview

The CareerForge Minimum Viable Product (MVP) focuses on creating a functional job discovery and recruitment platform that connects job seekers with employers.

The goal of the MVP is to solve the core problems:

- Helping job seekers discover suitable opportunities.
- Helping employers publish jobs and find candidates.
- Providing a simple and organized recruitment process.

The MVP includes only the essential features required to validate the platform's main purpose.

---

# 2. User Roles

CareerForge MVP supports three primary user roles:

## 1. Guest User

A visitor who can explore available public content without creating an account.

## 2. Job Seeker

A user looking for employment opportunities.

## 3. Employer

A company representative who publishes jobs and manages candidates.

*(Administrator features will be considered as a system management requirement and expanded later.)*

# 4. MVP Excluded Features (Future Versions)

The following features are intentionally excluded from MVP:

## Advanced Matching

- AI job recommendations
- Candidate matching score
- Skill gap analysis

## Communication

- Real-time chat
- Video interviews
- Messaging system

## Career Enhancement

- AI resume improvement
- Interview preparation
- Career guidance

## Platform Expansion

- Mobile applications
- Social networking
- Company reviews
- Advanced analytics

These features may be introduced in future versions after validating the core platform.

---

# 5. MVP Success Criteria

CareerForge MVP is considered successful when:

## Job Seekers can:

- Create professional profiles.
- Find relevant jobs.
- Apply for opportunities.
- Track applications.

## Employers can:

- Create company profiles.
- Publish job opportunities.
- Review applicants.
- Manage recruitment progress.

The MVP should demonstrate that CareerForge can improve the connection between candidates and employers.
